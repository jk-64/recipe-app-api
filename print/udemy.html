UPDATE: Docker Compose and Ubuntu Runner

As a number of students have pointed out in the Q&A, there are two updates to GitHub Actions which break the original code in the video.

These issues are:

  1. The docker-compose command has been removed, so you see the error: docker-compose command not found.

  2. The ubuntu-20-04 runner has been removed, which stops the job running.

This has already been fixed in the supplementary source code for the project. So if you used the diff link, then you will have the updated version.

For reference, the updated version of the .github/workflows/checks.yml file will look like this:

  1. ---
  2. name: Checks
  3.  
  4. on: [push]
  5.  
  6. jobs:
  7.   test-lint:
  8.     name: Test and Lint
  9.     runs-on: ubuntu-24.04
  10.     steps:
  11.       - name: Login to Docker Hub
  12.         uses: docker/login-action@v3
  13.         with:
  14.           username: ${{ secrets.DOCKERHUB_USER }}
  15.           password: ${{ secrets.DOCKERHUB_TOKEN }}
  16.       - name: Checkout
  17.         uses: actions/checkout@v4
  18.       - name: Test
  19.         run: docker compose run --rm app sh -c "python manage.py test"
  20.       - name: Lint
  21.         run: docker compose run --rm app sh -c "flake8"

This should fix both of these issues.
